Initial public release.

### Features

- Download entire repositories, folders, or single files from GitHub URLs
- Support for branch/tag/commit references
- `-o` flag for custom output directory or filename
- `--token` and `--gh` flags for authenticated access
- Progress display with spinners and download bars
